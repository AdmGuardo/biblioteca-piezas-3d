# Documentación general

Aquí se almacena documentación relacionada con el proyecto que no pertenece a una pieza concreta.

---

## ¿Qué va aquí?

- Memorias de proyectos
- Manuales de software (Fusion 360, PrusaSlicer, etc.)
- Guías de uso de impresoras
- Procedimientos de trabajo del departamento
- Recursos educativos generales

---

## ¿Qué NO va aquí?

La documentación técnica de una pieza específica (planos, fichas) va dentro de la carpeta de esa pieza en `piezas/`.
