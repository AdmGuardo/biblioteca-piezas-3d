# Imágenes 

Esta carpeta es para recursos gráficos generales (logos, banners...).

> LAS IMAGENES DE LAS PIEZAS PARA EL CATÁLOGO WEB NO VAN AQUI, VAN EN SU RESPECTIVA CARPETA SITUADA EN `piezas/`

---

## Formatos recomendados

| Formato | Cuándo usarlo |
|---------|---------------|
| `PNG` | Logos, capturas, imágenes con transparencia |
| `JPG` | Fotografías de piezas |
| `WEBP` | Si quieres optimizar el peso sin perder calidad |

## Recomendaciones

- Nombres en minúsculas y con guiones, sin espacios: `logo-principal.png`
- Resolución mínima recomendada para previews: 800×600 px
- Evita subir imágenes de más de 2 MB
