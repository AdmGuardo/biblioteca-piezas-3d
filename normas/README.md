# Normas de la biblioteca

## Nombrado de carpetas

Formato:

XXX-nombre-pieza

Ejemplo:

007-pinza-neumatica

## Archivos obligatorios

- ficha.md
- STL
- F3D

## Archivos recomendados

- Fotografías
- Planos técnicos
- Manuales

## Commits

Utilizar mensajes descriptivos.

Ejemplos:

Añadida pieza 007 - Pinza neumática

Actualizada documentación del soporte de motor
