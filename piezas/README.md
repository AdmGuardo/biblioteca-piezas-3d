# Piezas

Esta carpeta contiene las piezas desarrolladas por alumnos y profesores.

Cada pieza debe almacenarse en una carpeta independiente.

Ejemplo:

001-soporte-motor/
002-engranaje/
003-carcasa-sensor/

Cada carpeta debe incluir:

- ficha.md
- STL
- F3D
- Fotografías
- Planos técnicos (si existen)
